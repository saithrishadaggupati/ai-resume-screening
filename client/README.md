# \# RecruitAI — Frontend

# 

# This is the frontend client for \*\*RecruitAI\*\*, an AI-powered resume screening platform built for the Google Gen AI Academy APAC Cohort 2 Hackathon.

# 

# RecruitAI helps recruiters evaluate candidates in seconds instead of minutes. Upload a resume against a job posting and instantly receive a compatibility score, matched/missing skills, strengths, weaknesses, and tailored interview questions — powered by Google Gemini 2.5 Flash.

#

# \## Tech Stack

# 

# \- React 19 + Vite

# \- Material UI

# \- Recharts

# \- Axios

# 

# \## Features

# 

# \- Recruiter dashboard with candidate rankings and analytics

# \- Job posting creation and management

# \- PDF resume upload with drag-and-drop

# \- AI-generated compatibility scores, skill analysis, and interview questions

# \- Candidate search by name, skills, or experience

#

# \## Backend

# 

# This frontend connects to the RecruitAI backend (Node.js + Express, deployed on Google Cloud Run) for authentication, job management, resume uploads, and AI analysis. The backend API base URL is configured in \\`src/services/api.js\\`.

# 

# \## Google Cloud Services Used

# 

# \- \*\*Google Gemini 2.5 Flash\*\* — resume analysis and scoring

# \- \*\*Google Cloud Storage\*\* — secure resume PDF storage

# \- \*\*BigQuery\*\* — real-time hiring analytics

# \- \*\*Cloud Run\*\* — serverless backend hosting
